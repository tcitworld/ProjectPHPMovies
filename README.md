## HowTo

* `git clone https://github.com/tcitworld/ProjectPHPMovies`
* `composer update`
* `cd web`
* `php  -S localhost:8000`
* head over to `http://localhost:8000/`

## Features

* List all movies
* Edit a movie
* Delete a movie
* Add a movie
* See details for a movie
* Sort fields
* Search through rows
* Pagination

All through Ajax.
